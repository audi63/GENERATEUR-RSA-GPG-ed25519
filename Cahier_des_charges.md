# Cahier des Charges - Gestionnaire de Clés SSH et GPG

## Objectifs du Projet

- Simplifier la gestion des clés SSH et GPG pour les développeurs.
- Proposer une interface utilisateur moderne et intuitive.
- Automatiser des tâches répétitives et complexes.

---

## Fonctionnalités Clés

1. Création de clés SSH et GPG.
2. Suppression et gestion des clés existantes.
3. Intégration avec VS Code pour l'édition du fichier `config`.
4. Packaging en une application exécutable.

---

## Contraintes Techniques

- Utilisation de Python et PySide6.
- Interface moderne avec `qt-material` pour un thème sombre.

---

## Points à Améliorer

1. **Ouverture de fichiers avec VS Code**.
2. **Affichage des icônes dans les boutons**.
3. **Intégration avec GitHub pour automatiser l'ajout de clés publiques**.
